How to set up ntfy (free, 2 minutes):
  1. Download the ntfy app — iPhone: App Store, Android: Play Store
  2. Open the app and tap + to subscribe to a new topic
  3. Choose a topic name that is hard to guess (e.g. sarah-johnson-x7k2)
  4. Your webhook URL is: https://ntfy.sh/YOUR-TOPIC-NAME
  5. Paste that URL in the field below

Topic names are public — use a random or hard-to-guess name. Free tier: 250 notifications per day.

How to set up SimplePush (free, 2 minutes):
  1. Download the SimplePush app — iPhone: App Store, Android: Play Store
  2. Open the app — your unique key is shown on the main screen
  3. Your webhook URL is: https://api.simplepush.io/send/YOUR-KEY
  4. Paste that URL in the field below

Free tier: 10 notifications per month. $12.49/year for unlimited.

How to set up Pushover ($5 one-time, 30-day free trial):
  1. Download the Pushover app and create a free account
  2. Visit pushover.net/apps/build and register a new application — name it "RPR Leads" or similar, then copy the API Token
  3. Copy your User Key from the Pushover dashboard
  4. Your webhook URL is: https://api.pushover.net/1/messages.json?token=API_TOKEN&user=USER_KEY
  5. Paste that full URL in the field below

10,000 notifications per month. $5 one-time purchase per platform.

How to set up Toolkit.app (free, 2 minutes):
  1. Go to gettoolkit.app/tools/api/webhook-to-email
  2. Enter your email address and click Send Magic Link
  3. Click the link in your email to verify
  4. Copy the webhook URL provided
  5. Paste it in the field below

Completely free. No account required. Leads arrive as email.

How to set up Slack (free, 5 minutes):
  1. Go to api.slack.com/apps
  2. Click Create New App > From scratch
  3. Name it "RPR Leads" and pick your workspace
  4. Go to Incoming Webhooks and toggle it On
  5. Click Add New Webhook to Workspace
  6. Pick the channel where leads should appear
  7. Copy the webhook URL and paste it below

Free on all Slack plans. Leads appear as formatted channel messages.

How to set up Discord (free, 2 minutes):
  1. Open Discord and go to the channel where you want leads
  2. Click the gear icon (Edit Channel) next to the channel name
  3. Go to Integrations > Webhooks > New Webhook
  4. Name it "RPR Leads"
  5. Click Copy Webhook URL and paste it below

Completely free. No account setup beyond your existing Discord.

How to set up Google Sheets (free, 10 minutes):
  1. Go to sheets.new to create a new Google Sheet
  2. Name the first row columns: Timestamp | First Name | Last Name | Email | Phone | Area | Report URL
  3. Click Extensions > Apps Script
  4. Delete all existing code and paste the RPR script (see link below)
  5. Click Deploy > New Deployment > Web App
  6. Set "Who has access" to Anyone
  7. Click Deploy and authorize when prompted
  8. Copy the web app URL and paste it below

Full setup guide: blog.narrpr.com/google-sheets-setup. Free. Leads stored as rows. Enable email notifications in Tools > Notification Rules.

How to set up GoHighLevel:
  1. In GHL go to Automation > Create Workflow
  2. Add Trigger: Inbound Webhook
  3. Copy the webhook URL generated
  4. Paste it in the field below
  5. Add actions to notify yourself or create a contact

First 100 leads free, then $0.01 per lead. Requires an active GoHighLevel account.

How to set up Make ($9/month Core plan):
  1. Sign up at make.com and create a new Scenario
  2. Add a Custom Webhook as the trigger module
  3. Click Create a webhook and copy the URL
  4. Paste it in the field below
  5. Add actions to email yourself, add to a CRM, etc.

Make Core plan required for webhook triggers ($9/month).

How to set up Zapier (paid plan required):
  1. Sign up or log in at zapier.com
  2. Create a new Zap > Trigger: Webhooks by Zapier > Catch Hook
  3. Copy the webhook URL from the Test tab
  4. Paste it in the field below
  5. Add an action (email, CRM, Google Sheets, etc.)

Webhooks by Zapier requires a paid plan (from $20/month). When you click "Send test" above, Zapier may show the body as a single field — your live form submissions are sent as proper JSON and will be parsed normally.

Paste any HTTPS webhook URL below. The widget will POST lead data as JSON to this URL when a visitor submits the form.

Payload includes: first_name, last_name, email, phone, selected_area, report_url, timestamp, form_id.

Webhook URL must start with https:// — leads will not be sent over insecure connections.

Webhook URLs grant write access to your channel. Don't share the generator link publicly.
